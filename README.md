# Cálculo do Terceiro Ângulo de um Triângulo

Este programa em C calcula o terceiro ângulo de um triângulo a partir de dois ângulos fornecidos pelo usuário.

---

## Descrição do Problema

O programa deve:

* Receber dois ângulos de um triângulo
* Calcular o terceiro ângulo
* Exibir o resultado com duas casas decimais

**Regra importante:**
A soma dos ângulos internos de um triângulo é sempre **180°**.

---

## Lógica da Solução

1. O usuário informa dois ângulos (`a` e `b`)
2. O programa verifica se os valores são válidos
3. Calcula o terceiro ângulo:

```
c = 180 - a - b
```

4. Exibe o resultado formatado

---


## Conceitos Utilizados

* Entrada de dados com `scanf`
* Operações matemáticas simples
* Estruturas condicionais (`if`)
* Formatação de saída (`printf`)
